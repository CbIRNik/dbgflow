# dbgflow-macros

Procedural macros for the `dbgflow` graph debugger ecosystem.

Published package:

- crates.io: [crates.io/crates/dbgflow-macros](https://crates.io/crates/dbgflow-macros)
- docs.rs: [docs.rs/dbgflow-macros](https://docs.rs/dbgflow-macros)

This package exports:

- `#[trace]`
- `#[ui_debug]`
- `#[dbg_test]`

Most users should depend on `dbgflow` instead of wiring this crate directly.
